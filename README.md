## 安装步骤

1. 克隆到本地
   ```
   git clone git@github.com:Twitzz/vue3-manage-system.git
   ```
2. 进入目录
   ```
   cd vue-manage-system
   ```
3. 安装项目依赖
   ```
   yarn install
   ```
4. 启动项目，浏览器访问 http://localhost:8080
   ```
   yarn dev
   ```
5. 执行构建命令，生成的dist文件夹放在服务器下即可访问
   ```
   yarn build
   ```
